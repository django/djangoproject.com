{% translate "Supported Versions" %}

{% blocktranslate trimmed %}Feature releases (A.B, A.B+1, etc.) will happen roughly every eight months. These releases will contain new features, improvements to existing features, and such.{% endblocktranslate %}

{% blocktranslate trimmed %}Patch releases (A.B.C, etc.) will be issued as needed, to fix bugs and/or security issues. These releases will be 100% compatible with the associated feature release, unless this is impossible for security reasons or to prevent data loss. So the answer to "should I upgrade to the latest patch release?” will always be "yes."{% endblocktranslate %}

{% blocktranslate trimmed %}Certain feature releases will be designated as long-term support (LTS) releases. These releases will get security and data loss fixes applied for a guaranteed period of time, typically three years.{% endblocktranslate %}

{% blocktranslate trimmed %}See the supported versions policy for detailed guidelines about what fixes will be backported.{% endblocktranslate %}
