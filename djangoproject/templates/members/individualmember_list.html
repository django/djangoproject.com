{% blocktrans %} Individual Members are appointed by the DSF in recognition of their service to the Django community. Service to the Django community takes many forms. Here are some non-exhaustive examples of the categories of work that might qualify as service:

  • Code contributions on Django projects or major third-party packages in the Django ecosystem
  • Reviewing pull requests or triaging Django project tickets
  • Documentation, tutorials or blog posts related to Django
  • Discussions about Django in community areas such as the django-developers mailing list and Django Forum
  • Attending or organizing Django-related events or user groups
If you would like to apply for Individual Membership, please {% endblocktrans %} {% trans "fill out this form" %}.

{% blocktrans %} You can also nominate others if you know someone who should be considered. If you are unsure if you meet the criteria, but you would like to be a member, please apply. If you're not accepted, you will receive a response with information about how you can successfully apply in the future. {% endblocktrans %}
