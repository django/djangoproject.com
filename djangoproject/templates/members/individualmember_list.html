{% blocktranslate trimmed %} Individual Members are appointed by the DSF in recognition of their contribution to the DSF's mission of advancing and promoting Django, protecting the framework's long-term viability, and advancing the state of the art in web development. {% endblocktranslate %}

{% blocktranslate trimmed %} Contribution to the DSF's mission takes many forms. Here are some non-exhaustive examples of the categories of work that might qualify: {% endblocktranslate %}

  • {% blocktranslate trimmed %} Contributing code, documentation, or tests to Django or to major third-party packages in the Django ecosystem. {% endblocktranslate %}
  • {% blocktranslate trimmed %} Reviewing pull requests or triaging tickets on Django or a third-party app. {% endblocktranslate %}
  • {% blocktranslate trimmed %} Creating learning resources (blogs, videos, etc.) for people to learn Django. {% endblocktranslate %}
  • {% blocktranslate trimmed %} Contributing to discussions in community areas such as the Django Forum or Discord. {% endblocktranslate %}
  • {% blocktranslate trimmed %} Being part of the organizing team for a Django community event. {% endblocktranslate %}
  • {% blocktranslate trimmed %} Serving on a DSF Working Group. {% endblocktranslate %}

{% blocktranslate trimmed %} For more information about membership, see the DSF membership FAQ. {% endblocktranslate %}

{% blocktranslate trimmed %} If you would like to apply for Individual Membership, please fill out this form. If you are unsure if you meet the criteria, but you would like to be a member, please apply anyway! You can also nominate others using the same form if you know someone who should be considered. {% endblocktranslate %}

{% blocktranslate trimmed %} As a member of the DSF, you will be recognized for your contributions to the community. Your name will appear below and you'll be added to the various DSF Members communication channels (mailing list, forum, Discord). You will also be eligible to vote for the DSF Board and Steering Council. {% endblocktranslate %}
